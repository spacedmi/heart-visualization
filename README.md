heart-visualization
===================

Human heart visualization project

Heart's mesh partition using Metis
Run command example: ./mesh-partition heart.node heart.ele 2 1 1
Algorithms:
1 - PartGraphRecursive
2 - PartGraphKway
Weighted:
0 - unweighted graph
1 - weighted graph
